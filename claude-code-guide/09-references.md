## 学习路径建议

### 初学者
1. 阅读 [Claude Code Complete Guide 2026](https://www.jitendrazaa.com/blog/ai/claude-code-complete-guide-2026-from-basics-to-advanced-mcp-2/)
2. 观看入门视频教程
3. 尝试基本命令和工作流
4. 加入 Discord 社区提问

### 中级用户
1. 学习 [上下文工程](https://01.me/en/2025/12/context-engineering-from-claude/)
2. 掌握 [Subagent 系统](https://code.claude.com/docs/zh-CN/sub-agents)
3. 配置 [Hook 系统](https://code.claude.com/docs/en/hooks)
4. 创建自定义 Skills

### 高级用户
1. 开发自定义 MCP 服务器
2. 构建复杂的多代理工作流
3. 优化性能和 token 使用
4. 贡献开源项目和分享经验

### 企业用户
1. 学习安全和合规配置
2. 设置团队协作环境
3. 集成企业工具和服务
4. 制定最佳实践和标准
