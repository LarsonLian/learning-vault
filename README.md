# learning-vault

一个精心整理的开发者学习资源库，汇集了软件工程、系统架构、编程语言、工具链等领域的优质学习资料、实践经验和知识清单。

## 目录结构

### 📚 claude-code-guide/
Claude Code 开发指南完整文档集，涵盖架构、上下文工程、子代理系统等核心主题。

**主要文档：**
- `01-architecture.md` - 架构设计与核心概念
- `02-context-engineering.md` - 上下文工程最佳实践
- `03-subagent-system.md` - 子代理系统详解
- `04-async-agents.md` - 异步代理开发指南
- `05-advanced-features.md` - 高级特性使用说明
- `06-hook-system.md` - Hook 系统开发文档
- `07-mcp-integration.md` - MCP 协议集成指南
- `08-best-practices.md` - 最佳实践与编码规范
- `09-references.md` - 参考资料与扩展阅读
- `10-rules-system.md` - 规则系统设计文档
- `11-plugins-system.md` - 插件系统开发指南
- `12-agent-teams.md` - Agent Teams 多会话编排官方指南
- `13-context-engineering-fundamentals.md` - 上下文工程基础概念 ⭐ 新增
- `README.md` - 指南总览与导航

**子目录：**
- `开发agent/` - Agent 开发实战资源
  - `skills/` - 技能模块集合
  - `subagents/` - 子代理配置示例
  - `INSTALL.md` - 安装配置说明
  - `SYNC_LOG.md` - 同步日志记录
  - `README.md` - Agent 开发入门指南

### 📊 langgraph/
LangGraph 完整学习指南,涵盖从基础概念到生产部署的所有知识点。

**主要文档:**
- `LangGraph-Guide.md` - 15章完整教程(19万字深度指南)
- `LangGraph-QuickRef.md` - 快速参考速查手册
- `LEARNING-CHECKLIST.md` - 系统化学习检查清单
- `README.md` - 学习路径与章节索引

**核心内容:**
- **基础知识篇** (第1-4章): LangGraph 核心概念与快速上手
- **核心能力篇** (第5-10章): 状态管理、条件路由、并行执行、人机协作、持久化、模块化
- **实战模式篇** (第11-15章): 完整案例、架构模式、性能优化、错误处理、测试调试

**子目录:**
- `examples/` - 完整项目案例和代码示例

### 🤖 nanobot/
Nanobot 架构分析与技术文档。

**文档:**
- `2026-02-10-nanobot-architecture-deep-dive.md` - Nanobot 架构深度剖析

## 使用说明

1. **学习 Claude Code 开发**: 从 `claude-code-guide/README.md` 开始,按章节顺序阅读
2. **学习 LangGraph**: 从 `langgraph/README.md` 开始,根据学习路径选择对应章节
3. **开发 Agent**: 参考 `claude-code-guide/开发agent/` 目录下的实战资源
4. **了解 Nanobot**: 查阅 `nanobot/` 目录下的架构分析文档
